RFLCT An Intelligent Reflection.

A smart mirror utilizing voice detection and facial recognition to display real-time, customizable information based on who you are. Your personal profile will have weather, news feeds from any website, a personal calendar, and Amazon Alexa ready to answer any question you may have.

dotCHOMP Hunger. Hunger Never Changes.

This was a personal project built during my time at a coding bootcamp. A HTML5 browser game for desktop browsers. You are a young mushroom trying to live in a famished environment. How long can you survive a voracious appetite?

Application Database On Track. All the Time.

Constructed for a State Department to track applications submitted by thousands of companies. Using Microsoft Access and Visual Basic, this generic named database is still used today for applications that have a total value of over $5 million.
